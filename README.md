# Amazon-Sales-Performance-Dashboard
## Overview
This project analyzes Amazon sales data using Microsoft Excel and presents key business insights through an interactive dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## KPIs
- Total Revenue: $84.98M
- Total Orders: 100,000
- Total Customers: 43,233
- Average Order Value: $850
- Total Quantity Sold: 300,140
- Delivered Order Rate: 74.63%

## Dashboard Features
- Monthly Sales Trends
- Category Wise Sales
- Top 10 Products
- Country Wise Revenue
- Payment Mode Analysis
- Order Status Analysis
- Interactive Filters

## Key Insights
- Electronics generated the highest sales.
- United States contributed the highest revenue.
- 75% orders were delivered successfully.
- Cancellation and return rates remained low.
- Credit Card was the most preferred payment method.
